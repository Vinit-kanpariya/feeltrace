## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-03-05 | Tampering | Prompt injection via signal values | accept | ScoredIssue fields are numeric or short enum strings from deterministic scoring — no raw page content reaches the LLM |
| T-03-06 | Tampering | LLM inventing issue indices | mitigate | parseStage2Output discards enriched_issues with index >= scoredIssues.length; validated by unit test |
| T-03-07 | Tampering | LLM inventing mechanism strings | mitigate | z.enum(PERMITTED_MECHANISMS) rejects any mechanism not in the 13-item list; zod throws before DB write |
| T-03-08 | Denial of Service | LLM cost overrun on Stage 2 | mitigate | max_tokens: 2048 cap per call bounds spend; INFRA-02 upstream rate-limits submissions per IP |
| T-03-09 | Information Disclosure | ANTHROPIC_API_KEY logged | mitigate | Key is read from env and passed to SDK constructor — never interpolated into strings, never logged, never returned in API responses |
| T-03-SC | Tampering | npm/pip/cargo installs | mitigate | slopcheck run on @anthropic-ai/sdk (2026-05-26) — [OK] verdict; no blocking checkpoint required |
